# Phase 3 — Meta-Analysis

Cross-study meta-analysis will add:

- Recurring DE genes across studies
- Recurring stress-response pathways
- Stress-response atlas
- Candidate biomarker ranking

## Planned Components

- New job type: `job:metaanalyze`
- Tables: `gene_occurrences`, `pathway_occurrences`, `biomarker_candidates`
- Pipeline: `pipelines/meta/` or `watchtower/meta/`
- AI `InterpreterProvider.suggest_biomarkers()` integration

Weekly digest will include a "recurring stress genes" section.
